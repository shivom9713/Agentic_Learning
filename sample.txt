Create a QA Test Case Execution skill that works for any generic application, where the application name is provided as <app_name>_app (for example, ./calculator_app); the skill must reference the ./test_cases folder, locate the corresponding test case file named <app_name>.xlsx (for example, calculator.xlsx), and execute all test cases defined within that file against the application, capturing execution status, actual results, and defects if any.
